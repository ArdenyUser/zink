Web Installer Readme

HOW TO INSTALL:

Step one: sudo apt-get update -y
Step two: sudo apt-get install -y links2

WEBINX-UBUNTU
(To use run the command after you install it, "links2 -g")

OS recommended for use:
The OS for this webinx installer to work on is Ubuntu. Ubuntu is what this is made for.
